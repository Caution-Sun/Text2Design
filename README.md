# Text2Draw
캡스톤 디자인 팀 You &amp; AI app UI 구현

***

## 개발 환경
IDE : Android Studio

Language : Java

***

기본적으로 pager을 통해 각 화면을 슬라이드로 전환할 수 있도록 만들었다.
